Advanced AR Teaching Tool

This version uses one shared scene. You upload several GLB files, place them in different positions in the same scene, add tap actions, and then preview the designed scene and interactions.

How it works
1. Upload several GLB files.
2. Add selected models into the empty scene.
3. Drag them or edit their position values.
4. Assign tap actions such as move, hide, or toggle.
5. Switch to Preview mode and test the scene interactions.
