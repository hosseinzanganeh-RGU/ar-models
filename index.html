Advanced AR Teaching Tool

This version creates one true 3D scene. You upload several GLB files, insert them into a shared 3D space, set their x, y, z positions, add actions, and then preview the same scene. On supported mobile devices, the AR mode lets you place the whole scene onto a detected floor or table and supports tap actions directly inside AR mode.

Important note
Real-world AR preview uses WebXR hit testing to place the whole scene on a detected surface such as a floor or table. It works best on Android Chrome and other WebXR-supported browsers. iPhone Safari usually does not support this full multi-object WebXR workflow. For mobile AR, this HTML file must be hosted over HTTPS such as GitHub Pages.
